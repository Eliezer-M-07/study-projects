# Formulário de Endereço com API de CEP

Projeto em JavaScript puro que consome a API ViaCEP para preenchimento automático de endereço.

## Tecnologias
- HTML
- CSS (Bootstrap)
- JavaScript (Fetch API)

## Funcionalidades
- Busca automática de CEP
- Preenchimento de endereço
- Validação de CEP
- Tratamento de erro

## API utilizada
https://viacep.com.br/
