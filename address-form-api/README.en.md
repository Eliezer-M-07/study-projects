# Address Form with CEP API

A pure JavaScript project that consumes the ViaCEP API to automatically fill in address fields.

## Technologies
- HTML
- CSS (Bootstrap)
- JavaScript (Fetch API)

## Features
- Automatic CEP lookup
- Address auto-fill
- CEP validation
- Error handling

## API Used
https://viacep.com.br/
