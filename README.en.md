# Study Projects

This repository contains a collection of personal projects developed during my programming studies. The main goal is to reinforce learning through practice by building small experiments with different technologies, concepts, and ideas.

## About

The projects in this repository are exploratory in nature and do not follow a specific theme or production goal. Each one is developed as part of my continuous learning process, based on curiosity, practice, or the need to understand a particular concept in depth.

Each folder represents an independent project, varying in scope, complexity, and technology stack.

## Objective

- Strengthen programming fundamentals through hands-on practice  
- Explore different tools, libraries, and frameworks  
- Experiment with new ideas and technical concepts  
- Improve problem-solving and development skills over time  

## Note

This repository is not intended for production use. It serves as a personal learning space to document progress and experimentation throughout my development journey.
